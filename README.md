# Secure Messenger — защищённый мессенджер с E2E-шифрованием

Курсовой проект по дисциплине «Методы и технологии программирования».
**Вариант 5:** реализация протокола безопасного обмена сообщениями
(X3DH + Double Ratchet, Perfect Forward Secrecy).

## Архитектура
- `secure_core/` — Rust-ядро криптоопераций (через PyO3)
- `client/` — Python-клиент: логика Double Ratchet + GUI (PyQt)
- `server/` — FastAPI релей-сервер («почтовый ящик», не расшифровывает сообщения)
- `docs/` — диаграммы (UML/sequence) и документация

## Технологический стек
Rust (x25519-dalek, chacha20poly1305, hkdf, zeroize) + Python (PyQt, FastAPI) + Docker.

## Свойства безопасности
- **PFS (Perfect Forward Secrecy)** — компрометация текущего ключа не раскрывает прошлые сообщения.
- **Post-compromise security** — система «самовосстанавливается» после утечки на следующем обмене.
